# 组件文档规范

## API类型说明

|    说明      |    类型    |
|    ---      |    ---     |
|    布尔      |    Boolean                                                   |
|    数字      |    Number                                                    |
|    字符      |    String                                                    |
|    数组      |    [] \| Array \| Array<Number> \| Array<String>             |
|    对象      |    {} \| Object                                              | 
|    枚举      |    Enum {Red, Green, Blue}                                   |
|    未知      |    any                                                       |
|    空值      |    null                                                      |
|    函数      |    (name: String) => Boolean                                 |
|    正则      |    RegExp                                                    |
|    React节点 |    ReactNode                                                 |
|    HTML节点  |    HTMLElementNode                                           |
