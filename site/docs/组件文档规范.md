## API类型说明

|    说明      |    类型    |
|    ---      |    ---     |
|    布尔      |    boolean                    |
|    数字      |    number                     |
|    字符      |    string                     |
|    数组      |    number[]\|string[]\|array  |
|    对象      |    object                     | 
|    枚举      |    enum {Red, Green, Blue}    |
|    未知      |    any                        |
|    空        |    null                       |
